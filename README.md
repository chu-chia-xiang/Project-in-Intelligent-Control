# 智慧型控制期末專題  
Adaptive Fuzzy Iterative Learning Control for High-Speed Trains With Both Randomly Varying Operation Lengths and System Constraint


## 研究背景與問題描述

高速列車（High-Speed Train, HST）在實際運行時，存在以下特性與問題：

- **非線性且重複的運行過程**：同一路線每天重複運行，但動態為非線性，且受外部擾動影響。
- **運行時間隨機變動**：每一趟列車實際運行長度 \(T_k\) 會在某範圍內隨機變動，而非固定值。
- **系統輸入與狀態具安全約束**：牽引力、制動力以及速度、位置等均需維持在安全範圍內。
- **外部擾動與不確定性**：風阻、坡度、負載變化等都會造成模型與實際系統間的偏差。


## 方法概要：AFILC + BCEF + 投影機制

論文提出的控制架構核心包含三個部分：

1. **Adaptive Fuzzy Iterative Learning Control (AFILC)**  
   - 利用模糊系統逼近未知的非線性列車動態。  
   - ILC 在每次運行後更新模糊參數與學習控制律，使軌跡追蹤誤差逐次減小。  

2. **Boundary Composite Energy Function (BCEF)**  
   - 建立一個包含狀態誤差與控制輸入的複合能量函數。  
   - BCEF 的設計考慮系統約束，透過分析 BCEF 的收斂性，保證在學習過程中：  
     - 軌跡追蹤誤差漸進收斂；  
     - 系統狀態與控制輸入不會超出預先設定的邊界。  

3. **Projection Mechanism（投影機制）**  
   - 當學習更新後的控制量或參數超出允許範圍時，會被投影回合法區間內。  
   - 確保參數估測及控制力始終滿足約束條件，避免數值發散或實際系統過載。  

整體而言，AFILC 在「迭代方向」利用 ILC 思想改善重複運行的性能，在「時間軸方向」則透過模糊近似與 BCEF 保證穩定性與約束滿足。

## 模擬設定

模擬環境依原論文設定進行，重點假設如下：:contentReference[oaicite:1]{index=1}

- **列車型號**：類似中國 CRH-3 型電力多單元列車  
- **總軌道長度**：約 52.3 km  
- **期望迭代時間**：名目值 \( T = 2500 \) 秒  
- **實際運行時間**：每一次運行 \(T_k\) 隨機取值於 \([2400, 2600]\) 秒  
- **質量**：\( M = 350{,}000 \) kg  
- **阻力模型**：包含基本滾阻與速度相關的附加阻力  
- **外部擾動**：隨機擾動項加入列車動態，用來模擬風阻、坡度或負載變化等不確定性  
- **控制目標**：  
  - 使列車速度與位置精準跟踪給定參考軌跡；  
  - 在所有迭代、所有時間內，控制力與速度等均滿足安全約束。
